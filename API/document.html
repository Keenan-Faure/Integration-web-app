API Documentation

Overview

  • JSON based REST-ful API that connects to a MySQL Database situated on the host machine.

Requirements

  • User must set Authorisation to Basic Auth and have valid credentials with which they use to connect to the Database.
  • User must set up the BaseURL in the Variables section of this collection and change the Current Value to the respective IP address.

Available Method Types

  • GET
  • POST

Available Endpoints

  • GET Utility
  • GET Products
  • GET Client

Products

Folder containing Product related endpoints such as

  • GET Products
  • Get ProductsBySKU
  • Get Count

Functionality

  • GET Products - Returns a maximum of 15 Products from the Inventory Table
  • Get ProductsBySKU - Returns a specific product queried by the SKU
  • Get Count - returns the amount of products in the Inventory Table

Customers

Folder containing Customer related endpoints such as

  • GET Customers
  • Get CustomerByID
  • Get Count

Functionality

  • GET Customer - Returns a maximum of 15 Customers from the Client Table
  • Get CustomerByID - Returns a specific product queried by the ID
  • Get Count - returns the amount of customers in the Client Table

Utility

Folder containing Utility endpoints

  • GET checkConnection
  • Get viewLog
  • Get viewTables

Functionality

  • GET checkConnection - Returns the current session connection
  • Get viewLog - Returns the current Log of the session.
  • Get viewTables - eturns all the tables in the current Database; First database is selected as the primary one.
